# Pokes Frontend

## Installation

To get started with the Pokes frontend, follow these steps:

1. **Clone the repository**

    ```bash
    git clone https://github.com/yourusername/pokes-frontend.git
    cd pokes-frontend
    ```

2. **Install dependencies**

    ```bash
    npm install
    ```
